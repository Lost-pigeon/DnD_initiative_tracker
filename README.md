# D&D Initiative Tracker / Трекер инициативы D&D

Онлайн-генератор двусторонних карточек инициативы для Dungeons & Dragons.

🌐 **[lostpigeon.ru](https://lostpigeon.ru)**

---

## Возможности

- Загрузка и кадрирование портрета персонажа
- Поля: имя, класс доспеха (КД), скорость
- Настраиваемая ширина карточки от 20 до 50 мм
- Двусторонняя карточка: верхняя половина перевёрнута на 180° для печати и сгиба по пунктирной линии
- Режим мастера — имя и характеристики только на одной стороне, вторая остаётся чистой для подписи от руки
- Поле для держателя-подставки с регулируемым размером
- Экспорт в PDF (A4 landscape), карточки автоматически укладываются по страницам
- Редактирование уже добавленных карточек, включая повторное кадрирование фото

## Стек

Полностью клиентское приложение.

| Что | Чем |
|-----|-----|
| Разметка | HTML5 |
| Стили | CSS (Custom Properties, CSS Grid, Flexbox) |
| Логика | Vanilla JS (ES2020+) |
| Шрифты | Google Fonts: Cinzel, Inter |
| Кадрирование фото | [CropperJS 1.6.2](https://github.com/fengyuanchen/cropperjs) |
| Рендер в PNG | [html2canvas 1.4.1](https://html2canvas.hertzen.com) |
| Генерация PDF | [jsPDF 2.5.1](https://github.com/parallax/jsPDF) |

Все три тяжёлые библиотеки (CropperJS, html2canvas, jsPDF) загружаются лениво — только в момент первого использования, что ускоряет первоначальный рендер страницы.

## Как запустить локально

Сайт состоит из статических файлов. Достаточно любого локального HTTP-сервера — открытие `index.html` напрямую из файловой системы не подойдёт из-за ограничений браузера на загрузку скриптов.

**Через Python:**
```bash
python3 -m http.server 8080
# открыть http://localhost:8080
```

**Через Node.js (npx):**
```bash
npx serve .
```

**Через VS Code:** расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).

## Как работает генерация карточки

### Геометрия

Все размеры карточки параметрически вычисляются через масштаб относительно базовой ширины 35 мм. При изменении ширины карточки все CSS Custom Properties обновляются динамически через `element.style.setProperty()`.

```
scale = cardWidth / 35
cardHeight = 120mm × scale
```

Двусторонняя карточка реализована в одном элементе: верхняя половина содержит зеркальную копию с `transform: rotate(180deg)`. При сгибе по пунктирной линии обе стороны смотрят наружу.

### Экспорт в PDF

1. Каждая карточка рендерится в скрытый DOM-элемент за пределами экрана
2. `html2canvas` делает снимок с `scale: 2` для чёткости
3. Укладчик `buildPdfPages()` строит строки карточек и укладывает их на листы A4 (landscape) с учётом отступов
4. `jsPDF` собирает многостраничный PDF и запускает скачивание

### Интернационализация (i18n)

Все строки интерфейса хранятся в `scripts/i18n.js` в виде объекта с ключами `ru` и `en`. HTML-элементы помечены атрибутами `data-i18n` (для текстового содержимого) и `data-i18n-attr` (для атрибутов, например `placeholder` и `alt`). Язык определяется автоматически из `navigator.languages` при первом визите, ручной выбор сохраняется в `localStorage`.
